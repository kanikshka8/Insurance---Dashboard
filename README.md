# Insurance Analytics Dashboard

##  Project Overview

This is a Power BI dashboard created to analyze insurance data such as premium amount, claim amount, and policy performance.

##  Key Features

* KPI Metrics (Premium, Coverage, Claims)
* Policy Type Analysis
* Active vs Inactive Policies
* Claim Status (Rejected, Settled, Pending)
* Age Group-wise Analysis
* Interactive Filters

## Tools Used

* Power BI
* DAX
* Power Query
